The Frontier gem has been written and bundled by Afri Schoedon (q9f).

It contains Dijkstra code derived from Jithin (jithinabraham) and Patrick Reynolds (piki).
- https://gist.github.com/piki/dc6a3ee8eb90be0f5c61dd972988094f

- https://gist.github.com/blurpesec/36a86540781e0d00f539067497e972db

It contains publicly available EVE Frontier Swagger API data for the test suite
which is subject to CCP Games' terms and conditions.
- https://docs.evefrontier.com/SwaggerWorldApi
